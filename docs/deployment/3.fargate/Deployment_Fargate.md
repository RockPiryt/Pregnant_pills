## Cloud-Native Roadmap

Planned architectural improvements:

- Migration to Amazon Aurora Serverless (PostgreSQL-compatible)
- Stateless application workloads on AWS Fargate
- Decoupling database from Kubernetes cluster
- Horizontal Pod Autoscaler (HPA)
- Observability stack (Prometheus + Grafana)
- Cost comparison between:
  - EC2 + k3s
  - EKS + managed nodes
  - EKS + Fargate
