# pregcare-web

Landing application for the PregCare platform.

## Responsibilities

- Serve the main entrypoint under `/`
- Present the platform overview
- Link users to domain-specific modules:
  - `/baby`
  - `/memo`
  - `/nutri`
  - `/org`
  - `/pills`

## Run locally

```bash
pip install -r requirements.txt
python app.py
